1-2:解决方案(服务器容器:tomcat):

1:请求头超过了tomcat的限值。本来post请求是没有参数大小限制，但是服务器有自己的默认大小。
2:配置 tomcat的server.xml文件,增加请求字段长度

增加maxHttpHeaderSize参数配置,当maxPostSize=0时,表示不限制.

1-3:SpringBoot项目:
